# How I Work

My design process is centered around understanding users and their goals before creating solutions. Throughout the semester, I followed a User-Centered Design approach that involved:

* Defining target users through personas
* Identifying user goals and tasks
* Creating storyboards and mockups
* Exploring alternative visual designs
* Developing interactive high-fidelity prototypes
* Applying visual communication principles
* Iteratively refining designs based on feedback

As I continue developing my UI/UX skills, I strive to create interfaces that are intuitive, visually appealing, and aligned with users' needs.

# Case Studies

## 1. Meridian Health Physiotherapy Website

### Live Website

https://meridianhealthphysiotherapy.netlify.app

### Overview

This project was developed as part of Assignment 2 and focuses on the design of a service business website for a physiotherapy clinic. The objective was to create a high-fidelity prototype that helps users access information about treatments, learn about available services, and book appointments efficiently.

The website combines the goals of multiple users into a single interface while applying User-Centered Design principles and visual communication concepts.

### Storyboard 1 – Sarah Copperfield

Storyboard Link:

https://www.figma.com/site/UO6Zdq5RqnbCo4HnXWVqRi/Storyboard-1---Sarah-Copperfield?node-id=0-1&t=jOhe6UrR9BIVLLyU-1

Sarah is experiencing knee pain and visits the website seeking treatment. She navigates to the Sports Injury Rehabilitation service page, reviews available appointment times, selects a time that fits her schedule, and successfully books an appointment.

### Storyboard 2 – Robert Martin

Storyboard Link:

https://www.figma.com/site/wekGbUESMJkhCsRPEQLj55/Storyboard-2---Robert-Martin?node-id=0-1&t=IEPg1uZEzHFqkg1H-1

Robert suffers from chronic back pain and visits the website to learn how physiotherapy can help improve his condition. He reviews treatment information and frequently asked questions before visiting the contact page to request additional information from the clinic.

### Key Features

* Service information pages
* Treatment descriptions
* Online appointment booking
* FAQ section
* Contact page
* Responsive design
* Consistent healthcare branding

### Technologies

* React
* JavaScript
* HTML5
* CSS3
* Bootstrap

## 2. Memory Game

### Status

In Progress

### Overview

The Memory Game is being developed as part of Assignment 3: *Games for Exploring Human Cognition*.

The objective of this project is to design an engaging game that helps users practice memory skills while incorporating concepts related to perception, attention, memory, and Gestalt principles. The project continues the User-Centered Design process by introducing personas, alternative storyboard designs, and an interactive high-fidelity prototype.

### Planned Features

* Multiple difficulty levels
* Customizable gameplay options
* Memory-based challenges
* Real-time feedback
* Performance tracking
* Responsive design
* Interactive gameplay

### Design Focus

* Human cognition
* Attention and perception
* Gestalt principles
* Visual hierarchy
* User engagement
* Interactive feedback

### Technologies

* React
* JavaScript
* HTML5
* CSS3
